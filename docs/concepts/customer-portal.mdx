The Customer Portal is a customizable web app you can white-label for your customers to install, monitor, and manage their own installs.

The portal lives under your subdomain and carries your branding.

![Branded Customer Portal](/images/concepts/customer-portal/branded-portal.png)

## What customers can do

From the portal, customers can:

- Browse the catalog of apps you publish to them
- Install an app into their own cloud account via CloudFormation / Azure Resource Group / GCP link
- Supply any `user_configurable` [inputs](/concepts/app-inputs) during install
- Approve provisioning and subsequent changes
- Monitor audit logs, health checks and install status
- See stack, sandbox, components, roles and policies

Vendors control which apps a customer sees, which inputs they can edit, and which approval style applies — one-click "approve all" versus step-by-step review.

## The vendor view

Vendors see customer-portal-created installs in the [dashboard](/concepts/dashboard) like a regular install.
